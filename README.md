# Ye Tian's GitHub Pages

个人静态网站，托管于 GitHub Pages。

## 项目列表

### 汉字学习 (hanz)
互动汉字学习工具，支持笔顺动画、部首筛选、笔画拆解、点阵可视化。

**功能特点**：
- 笔顺动画演示
- 部首筛选（按笔画分组）
- 笔画拆解与点阵可视化
- 生字本收藏
- PNG/SVG 导出

**访问地址**：[汉字学习](https://yetian.github.io/hanz/)

---

### 开发者工具箱 (devtools)

**访问地址**：[开发者工具箱](https://yetian.github.io/devtools/)

---

### 我的书架 (my_reading)
读书笔记项目，记录读过的书籍及其读后感。

**访问地址**：[我的书架](https://yetian.github.io/my_reading/)

---

### CVSS 漏洞评分计算器 (cvss)

**访问地址**：[CVSS 计算器](https://yetian.github.io/cvss/)

### 五行穿衣 (wxcy)

## 技术栈

- 纯 HTML + CSS + JavaScript（无框架）
- GitHub Pages 托管
- 无后端服务
- 数据存储在 localStorage

## 本地预览

```bash
# 使用任意静态服务器
python3 -m http.server 8080
# 或
npx serve .
```

然后访问 `http://localhost:8080`
