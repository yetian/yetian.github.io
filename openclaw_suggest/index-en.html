🔧 Security Issues from Misconfiguration

Config

Network Interface Exposure

AI Agent's WebSocket or API not bound to local address (127.0.0.1) may lead to remote code execution.

Mitigation: Bind to 127.0.0.1, use firewall rules
Config

Sensitive Credential Leak

API keys, tokens hardcoded or exposed via environment variables.

Mitigation: Use secret management services
Config

Excessive Permissions

AI Agent has more system permissions than needed.

Mitigation: Follow least privilege principle

📦 Supply Chain Security Issues

Supply

Malicious Packages

Fake GitHub releases or npm packages may contain malicious code.

Mitigation: Download from official repos, verify signatures
Supply

Dependency Chain Attack

Compromised open-source packages or outdated dependencies.

Mitigation: Update dependencies, use security scanners
Supply

Prompt Injection

Malicious websites can steal cookies via iframe or CSS spoofing.

Mitigation: Use separate browser profile

🧠 Issues from Misconceptions

Cognitive

Underestimating Technical Risk

Non-technical users may underestimate AI Agent risks.

Mitigation: Understand capabilities, seek guidance
Cognitive

Over-trusting Automation

Blindly trusting AI output without verification.

Mitigation: Always review AI-generated content
Cognitive

Ignoring Data Privacy

Entering sensitive data without considering risks.

Mitigation: Avoid sensitive data, use local deployment

💰 Cost-Related Issues

Cost

API Cost Overrun

Mitigation: Set daily/monthly spending limits

Mitigation: Evaluate total cost of ownership
