🔧 错误配置导致的安全问题

配置

网络接口暴露

AI Agent 的 WebSocket 或 API 接口未绑定到本地地址（如 127.0.0.1），可能导致远程代码执行。

防范：强制绑定到 127.0.0.1，使用防火墙规则限制访问
配置

敏感凭证泄露

API 密钥、Token 等敏感信息硬编码在代码中或通过环境变量意外暴露。

防范：使用密钥管理服务，避免在代码中存储凭证
配置

权限过度开放

AI Agent 拥有超出任务需求的系统权限，恶意指令可能造成更大损害。

防范：遵循最小权限原则，限制文件系统和网络访问

📦 供应链安全问题

供应链

恶意安装包

伪造的 GitHub 发布页或 npm 包可能包含恶意代码，已有多起开发者密钥被盗事件。

防范：仅从官方仓库下载，验证 GPG 签名，检查包完整性
供应链

依赖链攻击

通过入侵开源依赖包或使用过时、带漏洞的依赖来渗透您的系统。

防范：定期更新依赖，使用依赖安全扫描工具
供应链

提示词注入

恶意网页可通过 iframe 嵌入或 CSS 伪装窃取浏览器 Cookie，诱导执行未授权操作。

防范：使用独立浏览器配置，定期审查会话，限制敏感操作

🧠 错误认知导致的问题

认知

低估技术风险

非技术背景用户可能低估 AI Agent 的潜在风险，误以为工具"无风险"或"绝对安全"。

防范：充分了解 AI Agent 的能力边界，在专业人士指导下使用
认知

过度信任自动化

盲目信任 AI Agent 的输出，未验证代码安全性或操作后果。

防范：始终审查 AI 生成的代码和操作，保留人工审核环节
认知

忽视数据隐私

向 AI Agent 输入敏感业务数据或个人信息，未考虑数据泄露风险。

防范：避免输入敏感数据，使用本地部署版本或可信云服务

💰 成本认知导致的问题

成本

API 费用失控

未设置 API 调用限额，可能因 Prompt 过长、循环调用或恶意使用导致意外高额账单。

防范：在 API 提供商后台设置每日/每月消费上限
成本

隐性成本忽视

忽视学习曲线、调试时间、系统维护等隐性成本。

防范：评估总拥有成本（TCO），包括培训和维护费用
成本

安全事故成本

安全事故可能导致数据泄露、业务中断、声誉损失等远超技术本身成本的损失。

防范：将安全投入纳入预算规划，购买适当的安全保险
