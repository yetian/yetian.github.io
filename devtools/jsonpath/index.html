在左侧输入 JSON 数据，输入表达式后点击执行查询

常用表达式

  • $ — 根对象
  • $.store.book — 路径访问
  • $..author — 递归下降，获取所有 author
  • $.store.book[*] — 所有数组元素
  • $.store.book[0] — 第一个元素
  • $.store.book[-1:] — 最后一个元素
  • $.store.book[?(@.price < 10)] — 过滤表达式
