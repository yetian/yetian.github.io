什么是 Query、Key、Value？

类比：在图书馆找书

  • Query (Q)：你的需求 - "我想找关于XX的书"
  • Key (K)：书架标签 - 每本书的分类标签
  • Value (V)：书的内容 - 你最终获取的信息

当 Query 和 Key 匹配度高时，对应的 Value 就更重要。

注意力矩阵解读

  • 行 (i)：第 i 个词"看向"其他词的程度
  • 列 (j)：第 j 个词被其他词关注的程度
  • 颜色亮度：注意力权重越高，颜色越亮
  • 对角线：词对自身的注意力（通常较高）

为什么需要多层多头？

多层 (Layer)

每层学习不同层次的抽象。底层学习语法，高层学习语义。

多头 (Head)

每个头关注不同类型的关系：语法、指代、位置等。

💡 点击任意小方块查看该层该头的详细注意力

多头注意力 (8 heads)

为什么要"多头"？ 就像用多只眼睛看世界，每只眼睛关注不同方面。

局部注意力

关注相邻词语，捕获短语结构

全局注意力

均匀关注所有词，理解整体语义

前向注意力

关注前面的词，适合因果推理
